Amazon Selection Details
1st Round :- Online Written Technical Test

It included :

  1. 20 Multiple type Questions
  2. 2 programming Questions

For Correct Answer you will be given 1 mark and for wrong answer -0.25 will be deducted.

Questions on programming will be based on :

  1. Write a program to find the difference between the sum of nodes at odd height and the sum of nodes at even height.
  2. Given an array of integers representing coin values and the sum required. Find the number of coins required to get the sum.
2nd Round :- Technical

There are total 4 Technical Rounds. After each round candidates will be eliminated. Basically , questions are based on data structures. Technical Round lasts up to 60 to 90 mins.

Technical Questions that are basically asked are :

  1. Write a program to traverse the tree in spiral form in O (n) time. (Hint : use two stacks)
  2. Program to implement atoi function
  3. Program to swap the kth node from end and kth node from front
  4. Program to find loops in linked list
